## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- Modern web browser
- Network monitoring permissions (for production deployment)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/cybershield-ids.git
   cd cybershield-ids
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Access the dashboard:**
   Open your browser to `http://localhost:5173`

### Production Deployment

1. **Build the application:**
   ```bash
   npm run build
   ```

2. **Deploy to your server:**
   ```bash
   npm run preview
   ```

For production deployments, ensure proper network interface access and system permissions for real-time monitoring.
